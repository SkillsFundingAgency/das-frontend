Table cell width classes

CSS classes for applying percentage widths to table cells. Classes increment by 5%

  • das-table-cell-width-5 - applies width: 5%
  • das-table-cell-width-10 - applies width: 10%
  • das-table-cell-width-15 - applies width: 15%
  • ...
  • das-table-cell-width-90 - applies width: 90%
  • das-table-cell-width-95 - applies width: 95%
