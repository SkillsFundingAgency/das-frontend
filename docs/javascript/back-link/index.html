--- layout: javascript title: Back link current: back-link ---

A standard back link which is inserted into the page with JavaScript. The back link acts the same as the browser back button, and will go back to the previous page in the users browsing history when clicked.

The JavaScript will completely replace the contents of the element with the class .das-js-back-link, which allows you to define a fallback for when JavaScript is not enabled for the user.
