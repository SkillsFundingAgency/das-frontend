Selection of helpful classes

Class name Description
das-breakable Forces word to wrap, useful for long strings such as email addresses
das-no-wrap Will force a group of text not to wrap. Useful when makes sense for text to appear on the same line, like a date
das-js-hide Hide an element if JS is enabled
das-js-show Show an element (applies display: block) if JS is enabled
das-js-show--inline Show an element (applies display: inline) if JS is enabled
